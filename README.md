#RADOC/SICAD+

### Em 2023, a UFG continua usando o software **SICAD+**[^1]:<br>
&#x26BE; O SICAD+ é utilizado pelo docente APENAS para **consultar** o RADOC.<br>
&#x26BE; Não é possível alterar o RADOC via a _interface_ de usuário do SICAD+.
